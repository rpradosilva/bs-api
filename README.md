## About

- To use, access [JSON File](http://www.github.com/rpradosilva)

JSON structure example:

```json
{
  "id": 0, // auto id
  "toy_code": "", // string
  "toy_image": "", // string src
  "year": 0, // number
  "model": "", // open_roof or closed_roof
  "series": "", // string
  "external_color": "", // lowercase string
  "inner_color": "", // lowercase string
  "tampo": "", // string
  "details": "", // string
  "wheel_type": "", // string
  "madein": "", // string
  "notes": "" // string
}
```
